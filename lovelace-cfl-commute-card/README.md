# Lovelace CFL Commute Card

**Note:** This is a placeholder for the Lovelace card component.

The custom Lovelace card for displaying CFL Commute information should be created as a separate repository: `lovelace-cfl-commute-card`

## Features to Implement

- Display commute status with icon
- Show next train details (time, platform, delay)
- List upcoming trains
- Highlight disruptions
- Theme-aware styling

## Installation

After the card is created, install via HACS (Frontend category) or manually copy to `www/community/lovelace-cfl-commute-card/`

## Example Usage

```yaml
type: custom:cfl-commute-card
commute_name: Morning Commute
```
